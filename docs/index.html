The Entire API

Yes, really. That's all of it.

Open a file or folder

file = syft_perm.open("data.csv")

Grant permissions

file.grant_read_access("bob@university.edu")
file.grant_write_access("alice@company.com")
file.grant_admin_access("admin@org.com")

Check permissions

if file.has_read_access("user@example.com"):
    data = file.read()

Make public (carefully!)

file.grant_read_access("*")  # Anyone can read

How It Works: Managing syft.pub.yaml

SyftPerm automatically creates and updates your permission files:

# Your Python code:
project = syft_perm.open("research_data/")
project.grant_read_access("reviewer@journal.org")
project.grant_write_access("team@university.edu")

# Creates this syft.pub.yaml:
"""
rules:
  - pattern: '**/*'
    access:
      read:
        - 'reviewer@journal.org'
        - 'team@university.edu'
      write:
        - 'team@university.edu'
"""

# No manual YAML editing needed!
